# EZ Intelligence      
Antd + React + Flask RESTful + Echarts + Sqlite  

## Features  
- Admin  
Speech Recognition (Voice to word / Word to Voice)  
Face Recognition  
- Tools  
Translator  
- Visualization  
Echarts(CandleStick)  
  
## Run  
cd frontend  
npm install  
npm run build  
npm run start  
  
cd backend  
output requirements: pipreqs ./  
install requirements: pip install -r requirements.txt  
python -m flask run  
  
Linux Shell  
apt-get/yum install ffmpeg
