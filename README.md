# EGGroup AI    
Antd + React CLI + Flask RESTful + Sqlite  

## Features  
Speech Recognition (Voice to word / Word to Voice)  
Face Recognition  
Translator  
WebRTC(VideoChat)  
  
## Deploy  
pipreqs ./  
  
## Run  
cd frontend  
npm install  
npm run build  
npm run start  
  
cd backend  
pip install -r requirements.txt  
python -m flask run  
  
Linux Shell  
apt-get/yum install ffmpeg
